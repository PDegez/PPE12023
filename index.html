Mini projet

D'une liste d'urls à un fichier html fonctionnel


Step by step

  1. récupérer des urls et les enregistrer dans un .txt
  2. écrire un script bash pour lire les urls
    • ouvrir les urls avec curl
    • extraire les retours serveur et les encodages
    • renvoyer ces informations sous forme de données tabulées
  3. écrire un script bash pour transformer les données tabulées en tableau htlm
  4. génerer un site sur github à partir d'un document html
  5. faire du tableau html une page secondaire liée à la page d'accueil du site
